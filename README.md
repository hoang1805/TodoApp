# Hướng dẫn triển khai
## 1. Yêu cầu môi trường
- Node.js >= 16  
- npm hoặc yarn  
## 2. Cài đặt môi trường
### 2.1. Node.js
- Tải về và cài đặt Node.js từ [nodejs.org](https://nodejs.org/en/download) (nên dùng bản LTS).  
- Kiểm tra cài đặt:
  ```bash
  node -v
  npm -v
  ```
## 3. Triển khai ứng dụng
### 3.1. Clone từ github
```
# Clone project
git clone https://github.com/hoang1805/TodoApp.git
cd TodoApp

# Cài đặt dependencies
npm install
```
### 3.2. Khởi chạy
```
# Chạy server backend
npm start
```
### 3.3. Xem danh sách API
- Truy cập vào URL: `http://localhost:3000/docs`
